Primeiro Projeto: Lessons learned!

Ensinamentos Trybe atá o momento!

Primeiro Bash, usado no terminal. Então temos o Git, uma plataforma de versionamento de código. Nos ajuda a ter um maior controle sobre o código conforme adicionamos mais conteúdo. Temos também o GitHub, um repositório remoto onde esses códigos são salvos. Então ao HTML e CSS, usados para a criação de sites. O Html é uma linguagem de marcação de texto, responsável pelo conteúdo visual que queremos na página. O CSS por sua vez é responsável pela estilização da página.

Uma breve passagem

Esse primeiro projeto está se provando uma verdadeira aula! Acabei aprendendo mais coisas enquanto buscava resolver problemas.
